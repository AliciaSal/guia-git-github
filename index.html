Configuración

  1. git config --global user.name "tu_nombre"
  2. git config --global user.email "tu_correo@tucorreo.com"

Vincular la PC con Github

Crear la clave SSH

  1. ssh-keygen -t ed25519 -C "tu_correo@tucorreo.com"
  2. después de apretar ENTER nos va a preguntar donde guardarla
  3. Lo dejamos así como está para guardarla en la carpeta por defecto.
    Damos ENTER y nos pregunta si queremos ponerle un password.
  4. Le ponemos un password o dejamos como está y presionamos ENTER.
    Nos pide la confirmación y otra vez ENTER
  5. Abrimos la clave y copiamos la linea completa desde ssh hasta que finaliza el mail.
  6. Por último vamos a la página de Github y en nuestro avatar vamos a settings
  7. En setting entramos en SSH and GPG Keys y añadimos una clave con el botón verde.
  8. Le ponemos un título cualquiera y pegamos la clave copiada de la consola.
    Finalizamos con el botón Add SSH key y ya tendremos vinculada nuestra PC con Github
