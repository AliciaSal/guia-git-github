Comandos básicos

Vinculando el dispositivo con Github

Verificar si ya tenemos una clave ssh existente con la consola

En Windows (~ = Alt+126) y Mac (Hay una tecla ~ y `) :
ls -al ~/.ssh

Crear una Clave SSH: En Windows y Mac:
ssh-keygen -t ed25519 -C "your_email@example.com"
ENTER
Ponerle una "contraseña" o ENTER y repetirla cuando lo pida.
Abrir el archivo con un editor de texto y copiar la clave generada incluyendo el mail, o mostrar la clave con el comando
cat ~/.ssh/id_ed25519.pub
Copiar la clave (incluyendo el mail) y pegarla en github.
